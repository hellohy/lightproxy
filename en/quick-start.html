# Introduction


LightProxy is a whistle Electron based Proxy & Debugger Tools.


# What can LightProxy help you


  • Caputure network request & response
  • Real-time hosts binding
  • Forward remote url to local file or other remote file
  • mock interface and pages
  • Modify request & response, for example insert <script> , modify response header etc.


# Install

Open DMG ，Drag LightProxy to Application


# Start


Open LightProxy in Launchpad，First time LightProxy will ask for user password twice for install proxy helper and cert .


#


# Use / to Install Rule


Click  to create new rule，just input / to insert your rule.


For example we select Mock JSON



We will insert JSON rule with snippet。`` here which like string-literial in ES6 is a expand gammer based on  whistle , we can write multiple in-line files with this gammer.


Then we open browser to visit https://www.github.com/alibaba/lightproxy



We can see response content is JSON we wrote, my-test-head is also in response header ，meanwhile some helper header such as _ lightproxy-real-url_ can help us to location the real file source.


# More rule


Except for multiple line gammer, LightProxy just use gammer from whistle , you can just refer document of whistle : https://wproxy.org/whistle/principle.html
